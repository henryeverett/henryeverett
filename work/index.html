Pixel Burger

My first foray into indie game development, I created a simple-but-oh-so-fun physics based burger stacking game called Pixel Burger. Built on Apple's Sprite Kit framework and available for both iPhone and iPad, the game encourages you to stack an array of crazy ingredients as high as you possibly can without it cascading to the floor. All gameplay, art and assets were created solely by me.

It's the perfect one-minute, pick-up-and-play game, except we've been playing it all day. www.gamezebo.com

Moonlit

Built over the course of a weekend, Moonlit is a local events app for Bristol, UK. It displays details about upcoming music, comedy, art and film events pulled from a small Rails-based API and allows users to save them to their iPhone's calendar.
